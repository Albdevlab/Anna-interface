## Demo Access

During Phase 0, the ANNA interface will be accessible
through a public web endpoint.

The demo will allow reviewers to:
- Access the ANNA web interface
- Connect a Qubic wallet in read-only mode
- View basic wallet information
- Observe the Pulse-based system state UI
- Access introductory Academy content

---

## Verification Method

Phase 0 verification does not require:
- Any protocol-level interaction
- Any transaction signing
- Any economic or burn mechanisms

Verification can be performed by:
- Reviewing the public repository
- Accessing the live demo endpoint
- Checking documentation and acceptance criteria
